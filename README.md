## Project setup
### Python requirements
Are contained in `requirements.txt`

tip on how to output requirements: `pip freeze > requirements.txt`

### venv setup example
```
python -m venv .\venv\the_nature_of_code
.\venv\the_nature_of_code\Scripts\activate
pip install wheel
pip install raylib
```
